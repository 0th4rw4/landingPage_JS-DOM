Acerca de esta web

Bienvenidos

Este es un texto de bienvenida del HTML que representa al tercer Parcial de la materia Programación I, en la pauta que corresponde al manejo de DOM y BOM.
En el mismo, se debe formatear el documento HTML sin alterar ninguna propiedad del mismo, desde el documento parcial_3.js que se entrega con este mismo archivo.
Todas las modificaciones se deben realizar según lo enseñado durante la cursada con el manejo de Document Object Model y/o Browser Object Model.

Pautas del TP

El documento ofrece tres areas a formatear, el primero corresponde a un div con id tabs, adentro del mismo se encuentra una colección de etiquetas DIV que representan distintas secciones.
Los DIVS tienen un elemento h3 y un parrafo
El alumno debe obtener los divs y crear un navegador de pestañas bajo el siguiente requerimiento:
- Cada div dentro del div #tabs es una pestaña distinta.
- El titulo de cada pestaña (la solapa clickeable) es el texto del h3.
- El contenido de cada pestaña es lo que se encuentre dentro del parrafo.

Funcionamiento

Al hacer click sobre una solapa, se debe mostrar el contenido del parrafo correspondiente al titulo clickeable.
Para que el alumno no tenga dudas, son 3 pestañas: Bievenidos, Pautas del TP, Funcionamiento
Debe tener presente la posibilidad de agregar más nodos [ DIV > h3 + p ].
